# 🔐 AUTHENTIFICATION

## ✅ REGISTER

**POST** /register

### Body

```json
{
  "name": "Test User",
  "email": "test@mail.com",
  "password": "password123",
  "password_confirmation": "password123"
}
```

### Attendu

* Status: 201
* Token retourné
* User créé en base

---

## ✅ LOGIN

**POST** /login

### Body

```json
{
  "email": "test@mail.com",
  "password": "password123"
}
```

### Attendu

* Status: 200
* Token retourné

---

## ✅ ME

**GET** /me

### Headers

Authorization: Bearer {TOKEN}

### Attendu

* User retourné

---

## ✅ LOGOUT

**POST** /logout

### Headers

Authorization: Bearer {TOKEN}

### Attendu

* Token supprimé
* Accès refusé après
